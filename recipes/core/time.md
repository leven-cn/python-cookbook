# Time

## Timestamp (UNIX Time)

```python
import time

# seconds
>>> t = time.time()
>>> isinstance(t, float)
True

# nanoseconds
>>> t = time.time_ns()
>>>isinstance(t, int)
True
```

## Local Time To Timestamp

```python
import time

>>> local_time = time.localtime()
>>> isinstance(local_time, time.struct_time)
>>> t = time.mktime(local_time)
>>> isinstance(t, float)
```

## String Format

```python
import time

>>> time.strftime('%Y-%m-%d %H:%M:%S', time.localtime())
'YYYY-MM-DD hh:mm:ss'
```
